📥 Install uv

Choose your platform — no Python required beforehand.

macOS / Linux

shell 
curl -LsSf https://astral.sh/uv/install.sh | sh

Windows (PowerShell)

powershell 
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

Via pip (any platform)

shell 
pip install uv

Verify the installation: uv --version

🗺️ Step-by-Step Tutorial

Create a new project

This scaffolds a project directory with a pyproject.toml and a default src/ layout.

shell 
uv init my-ai-project
cd my-ai-project

Pin a Python version

uv downloads and pins the requested Python version automatically — no pyenv needed.

shell 
uv python pin 3.12

Add dependencies

uv updates pyproject.toml and the lockfile in one step.

shell 
uv add openai httpx pydantic
uv add --dev pytest ruff mypy   # dev-only deps

Sync the virtual environment

Creates (or updates) .venv/ from the lockfile — fast and reproducible.

shell 
uv sync

Run your code

Use uv run to execute scripts inside the managed environment — no manual source .venv/bin/activate needed.

shell 
uv run python src/main.py
uv run pytest               # run tests
uv run ruff check .         # lint

Update & lock dependencies

Keep packages current while keeping the lockfile in sync.

shell 
uv lock --upgrade           # refresh uv.lock
uv sync                     # apply updates to .venv

🤖 Professional AI Project Structure

A recommended layout for AI / ML projects managed with uv:

my-ai-project/ ├── .python-version # pinned by `uv python pin` ├── pyproject.toml # deps, build config, tool settings ├── uv.lock # deterministic lockfile — commit this! ├── .venv/ # managed by uv — add to .gitignore ├── src/ │ └── my_ai_project/ │ ├── __init__.py │ ├── main.py │ ├── agents/ # LLM agent logic │ ├── prompts/ # prompt templates │ └── utils/ ├── tests/ │ └── test_main.py ├── notebooks/ # Jupyter exploration ├── .env.example # API key template └── README.md

Minimal pyproject.toml

toml 
[project]
name = "my-ai-project"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = [
    "openai>=1.0",
    "pydantic>=2.0",
]

[dependency-groups]
dev = ["pytest", "ruff", "mypy"]

[tool.ruff.lint]
select = ["E", "F", "I"]
